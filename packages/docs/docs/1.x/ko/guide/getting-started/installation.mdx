# 설치

:::warning 레거시 버전
1.x는 레거시 API입니다. 기존 1.x 통합을 유지보수하는 경우가 아니라면 최신 2.x 설치를 권장합니다.
:::

1.x 패키지를 설치합니다.

<PackageManagerTabs command="install react-native-youtube-bridge@1.x" />

## peer dependency

iOS와 Android에서는 `react-native-webview`가 필요합니다.

<PackageManagerTabs command="install react-native-webview" />

## 최소 조건

| 패키지                 | 버전       |
| ---------------------- | ---------- |
| `react`                | `>=16.8.0` |
| `react-native`         | `>=0.60.0` |
| `react-native-webview` | `>=11.0.0` |
