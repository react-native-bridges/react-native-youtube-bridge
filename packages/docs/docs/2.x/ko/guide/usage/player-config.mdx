# 플레이어 설정

초기 옵션은 `useYouTubePlayer`의 두 번째 인자로 전달합니다.

```tsx
const player = useYouTubePlayer('AbZH7XWDW_k', {
  autoplay: true,
  controls: true,
  playsinline: true,
  rel: false,
  muted: true,
});
```

## 주요 옵션

- `autoplay`
- `controls`
- `loop`
- `muted`
- `startTime`
- `endTime`
- `playsinline`
- `rel`
- `origin`
