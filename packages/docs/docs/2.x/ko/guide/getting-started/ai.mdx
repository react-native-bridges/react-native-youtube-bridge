# AI / LLM 요약

짧은 요약이 필요하면 아래처럼 이해하면 됩니다.

- `react-native-youtube-bridge`는 YouTube IFrame Player API를 React Native에서 쉽게 쓰기 위한 Hook 기반 래퍼입니다.
- 핵심 API는 `useYouTubePlayer`, `YoutubeView`, `useYouTubeEvent`입니다.
- source는 비디오 ID, `{ videoId }`, `{ url }` 형식을 받을 수 있습니다.
- 네이티브 플랫폼은 WebView를 사용하고, Web은 iframe API를 직접 사용합니다.
- 고급 시나리오에서는 `useInlineHtml={false}`와 `webViewUrl`로 외부 플레이어 페이지를 사용할 수 있습니다.
